## はじめに

「管理者パスワードをExcelで管理してる」「全サーバーのローカル管理者パスワードが同じ」——IT管理者の多くが抱える、特権アカウントのパスワード管理問題。PIMで解決できる部分とそうでない部分を整理し、PAMツールとの組み合わせで完全に解決する設計を考えます。

## 何が問題なのか

### 共有パスワード問題

```
悪い例:
・全サーバーのAdministratorパスワードが「Passw0rd!」で統一
・退職した人が知っている共有パスワードが変更されていない
・パスワードリストをTeamsのチャットで共有している
```

1台のサーバーから別のサーバーへの横移動（Lateral Movement）が容易になります。

### パスワードローテーション問題

手動でのパスワード変更は忘れられることが多く、変更した場合も全員への周知が必要になります。

## 解決アプローチの全体像

```
【レイヤーごとの管理ツール】
Entra IDクラウドアカウント（Global Admin等）
  → PIM（Just-in-Time）+ 条件付きアクセス

Windowsサーバー・クライアントのローカル管理者
  → LAPS（Windows LAPS）でパスワード自動ローテーション

Linux/ネットワーク機器・全般の特権アカウント
  → エンタープライズPAMツール（CyberArk / BeyondTrust等）
```

## Windows LAPS（ローカル管理者パスワードソリューション）

Microsoft純正の無料ツール。Windowsのローカル管理者（Administrator）パスワードを自動生成・定期ローテーションし、Entra IDまたはActive Directoryに安全に保管します。

```
【仕組み】
Windowsデバイス → 定期的にランダムなパスワードを自動生成
                → Entra IDのデバイスオブジェクトに暗号化保存

管理者が必要なとき:
  Entra IDポータルでデバイスを検索
  → 「ローカル管理者パスワードを表示」
  → 30〜60日で自動ローテーション
```

### Windows LAPSの設定

Intune + Windows LAPS ポリシーで展開します。

```
IntuneのエンドポイントセキュリティポリシーでLAPSを設定:
  ・バックアップ先: Azure AD（Entra ID）
  ・パスワード有効期間: 30日
  ・パスワードの複雑さ: 高
  ・管理者アカウント名: Administrator（またはカスタム）
```

## エンタープライズPAM：CyberArk / BeyondTrustの役割

Windows以外の特権アカウント（Linux root、ネットワーク機器、データベース）の管理にはエンタープライズPAMツールが必要です。

```
【PAMツールの主な機能】
・パスワードボルト: 特権パスワードを暗号化保管
・自動ローテーション: 定期的にパスワードを変更
・セッション録画: 特権作業の操作を録画・監査
・Just-in-Time: 必要なときだけセッション接続を許可
・Dual Control: 2人の承認が揃わないとアクセスできない
```

## EntraID + PIM + LAPS + PAM の全体設計

```
【クラウドID（Entra ID管理）】
Global Admin等の特権ロール
  → PIMでJust-in-Time管理
  → 条件付きアクセスでMFA・準拠デバイス強制

【Windowsローカル管理者】
Windowsサーバー・クライアント
  → LAPS（Windows純正）で自動ローテーション
  → Entra IDに保管（パスワードは管理者のみ閲覧可）

【その他の特権アカウント】
Linuxサーバー・DB・ネットワーク機器・サービスアカウント
  → CyberArk / BeyondTrust等のPAMツールで一元管理
  → セッション録画で全操作を監査
```

## 導入優先度の考え方

```
コスト低・効果高から始める:
  Step1: Windows LAPS 展開（無料・Intune統合）
  Step2: PIM 有効化（P2ライセンスが必要）
  Step3: PAMツール検討（コスト大だが高リスク環境では必須）
```

## まとめ

特権ID管理は「一つのツールで解決する」ものではなく、クラウドID・ローカル管理者・その他特権をレイヤーに分けて適切なツールを組み合わせるアーキテクチャ設計が重要です。まずは無料で始められるWindows LAPSとPIMの導入から着手しましょう。
